This project folder contains the necesssary code to run the SmartHomeDevice manager.
smartHome\src\main\java\org\raj\smartHomeDeviceManager\controllers : 
The controllers contain the API's required for the POST and GET functions for every function(Pub/sub/con)
smartHome\src\main\java\org\raj\smartHomeDeviceManager\domain
The domain folder contains the required methods for handling the data between the front end and the backend.
smartHome\src\main\java\org\raj\smartHomeDeviceManager\repos
The repository folder contains all the renvironment variables required to run the smartHome project
